# Checklist Portfolio

## Captures a produire

1. Hero de l'application avec bandeau systeme / LLM
2. Decision Summary en 5 cartes
3. Top insights + suggestions d'investigation
4. Chart principal + chart secondaire
5. Prediction engine avec top drivers
6. Simulation guidee avec avant / apres / delta
7. Decision engine avec recommandation scenario
8. Evidence Pack ouvert
9. Reponse du copilote avec prochaine action

## Sequence de demo recommandee

1. Charger la demo ventes
2. Montrer la synthese decisionnelle
3. Ouvrir une suggestion d'investigation
4. Entrainer le modele sur `revenue`
5. Lancer la simulation guidee
6. Preparer puis lancer le decision engine
7. Poser une question au copilote
8. Ouvrir l'evidence pack

## Message portfolio

Ce projet illustre la conception complete d'un AI Decision Copilot:

- design produit
- analytics metier
- moteur de recommandation
- couche LLM
- experience de demo client-ready

## Angle visuel

- interface premium et decision-first
- charge cognitive reduite
- separation claire entre signal, recommandation et preuve
- storytelling plus proche d'un outil SaaS que d'un notebook data
